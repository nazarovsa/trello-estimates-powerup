### How to Use

1. **Add Estimates**: Click on any card's "Estimate" badge to set a numeric value
2. **View on Cards**: Estimates appear as blue badges on the card front
3. **Calculate Totals**: Click the "..." menu on any list and select "Show Estimate Total"
4. **Edit Anytime**: Click the badge again to modify or clear the estimate
