Privacy Policy for Card Estimates Power-Up

Last Updated: January 18, 2026

Overview

Card Estimates is a Trello Power-Up that allows users to add numeric estimates to cards and calculate column totals. This privacy policy explains how the Power-Up handles data.

Data Collection and Storage

What Data We Collect

The Card Estimates Power-Up collects and stores:

  • Numeric estimate values entered by users for individual Trello cards
  • This data is stored using Trello's built-in storage API at the card level

What Data We DO NOT Collect

We do not collect, access, or store:

  • Personal information (names, email addresses, etc.)
  • Card content or descriptions
  • Board structure or organization
  • User activity or behavior
  • Any data outside of the numeric estimates you explicitly enter

How Data is Stored

  • All estimate data is stored directly in Trello's infrastructure using their official API
  • Data is stored as "shared" card data, meaning it's visible to all board members who have access to the card
  • No data is sent to external servers or third-party services
  • No data is stored outside of Trello's ecosystem

Data Access

  • Only users with access to a specific Trello board can view the estimates on that board
  • Estimates follow Trello's existing permission model
  • When you remove the Power-Up from a board, all estimate data remains in Trello's storage
  • You can clear individual estimates at any time using the "Clear" button in the estimate popup

Third-Party Services

This Power-Up does not:

  • Send data to any third-party analytics services
  • Use tracking cookies
  • Share data with advertisers
  • Integrate with external APIs (except Trello's official API)

Data Retention

  • Estimate data persists as long as the card exists in Trello
  • Deleting a card will delete all associated estimate data
  • Removing the Power-Up from your board does not automatically delete stored estimates

Your Rights

You have the right to:

  • View all estimates on your boards
  • Modify or delete any estimate at any time
  • Remove the Power-Up from your board
  • Request information about data storage (contact via GitHub issues)

Security

  • All data transmission uses HTTPS encryption
  • Data is stored using Trello's secure infrastructure
  • The Power-Up runs entirely in your browser and Trello's iframe environment
  • No server-side processing or storage

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date above.

Open Source

This Power-Up is open source. You can review the complete source code to verify these privacy practices.

Contact

For questions or concerns about privacy:

  • Open an issue on the GitHub repository
  • Review the source code to verify data handling practices

Compliance

This Power-Up:

  • Does not collect personal data as defined by GDPR
  • Does not use cookies
  • Does not track users
  • Operates entirely within Trello's data infrastructure

By using the Card Estimates Power-Up, you acknowledge that you have read and understood this Privacy Policy.
